Benji Savings App

About

Benji is a money saving app with a focus on meeting financial goals. With the app users are able to budget, save, and track expenses all in the same place. Benji aims to make saving for large purchases more accessible to those who are unfamiliar with the world of finance and can be used to save for any goal.

Problem Statement

Benji users are familiar with technology but are no experts in finance. So, it is important that the tool be effective, inviting, and simple to use. Data on income and expenses is clearly organized and easily recorded from a variety of devices.

User Stories

A user flow diagram was developed based on the three user stories below.

1. As a user, I want to see a dashboard of my finances clearly and visually, so that I can see how much I am spending on what at a glance.

2. As a user, I need to be able to tell the tool what my savings goal is and how long I have to reach it, so that I can save accordingly.

3. As a user, I want to receive a personalized savings plan, so that I can save enough money to reach my goal in time.
